# 🎧 Spotify Data Analysis using Python

This project focuses on analyzing Spotify music data to understand trends, patterns, and insights using Python.

---

## 📊 Project Overview

In this project, I performed data analysis on a Spotify dataset to explore:

- Popular songs and artists
- Relationship between features like energy, danceability, and popularity
- Trends in music data

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📁 Dataset

The dataset used in this project is included in this repository as:

- `spotify_data.csv`

---

## 🔍 Steps Performed

1. Data Cleaning  
2. Data Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization  

---

## 📈 Key Insights

- Songs with higher energy and danceability tend to be more popular  
- Some artists dominate the dataset in terms of popularity  
- Different features of songs influence listener preferences  

---

## 🚀 Conclusion

This project helped in understanding music trends and how different song attributes impact popularity. It also strengthened my skills in data analysis and visualization using Python.
